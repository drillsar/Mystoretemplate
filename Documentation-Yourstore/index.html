Installation Guide

Note : You are advised to read the installation procedure carefully, backup your store files and database first before installation or making any changes.
Note : Please Create the Database before Installation. The name of Database will be required when asked during Installation.

The theme can be installed in two 2 ways

  1. Quickstart Installation : If you want to test the theme and check how the live demo was configured, you can install demo containing the sample data and settings from our live demo.
  2. Theme Files Installation : If you already have a Zencart store and want to apply the theme to that store. For the store running Zencart v1.5.5b or v1.5.5a, please follow these steps. Please follow these steps if you are using Zencart v1.5.4 and want to install the theme to that store.

Quickstart Installation

  1. Quickstart Theme Installation is only for the users who want the latest Zencart version i.e 1.5.5f
  2. Unzip the main downloaded package from themeforest and find the Quickstart folder.
  3. Upload the content of Quickstart directory to your server using the FTP client or Cpanel
  4. Start installation by typing the URL of the directory where the Quickstart files have been placed.
    (Ex: localhost/ [directory name] /installation/index.php or domain name/ [directory name] /installation/index.php)
    The initial installation page should appear as normal Zencart installation and then follow the next steps as shown below :

Database Setup

This step requires the Database information for Installation. Image below shows the Steps mentioned.

  • Enter your Database Host Name. (Ex: localhost)
  • Enter your Database Username. (Ex: root)
  • Enter your Database Password. (As per your requirement)
  • Enter your Database Name. (Ex: zencart)
  • Leave Store Identifier (Table Prefix) field empty.
  • Click on "Continue". This will take you to the next step of installation process.

Admin Setup

  • Enter the admin information as per your requirement and click on Continue.
  • A temporary Admin Password will be created and the admin directory will be renamed automatically as per the new Zen cart installation process.

Setup Finished

  • This will complete the installation process and the page with both Admin and Store Front links will appear as shown below.
  • Now go to your installation directory and rename/delete the zc_install directory.
  • Click on Store Admin URL and it will ask to refresh the page again and a normal zen cart admin login page will open. Enter the Admin Username and Temporary password. This will prompt you to change the temporary password as per the normal zen cart installation process

Store Setup

  • Once logged into admin panel the process will ask you to enter the store details like as shown in the image below. Enter the required details and cick on Update button.

Install Related Products Module

  • Once you login to the Admin, a message on the top will be displayed stating Related Products not Installed.
  • Click on Click to Install link, which is shown in the image below. This will install the Related Products Module. Hence this completes the Quickstart Demo installation of the theme.


Theme Files Installation

  1. Unzip the main downloaded package from themeforest and find the YousrtoreThemePackageFiles folder.
  2. Rename the Your-Admin-Folder-Name directory with the name of your existing store admin direcctory.
  3. Now copy all the files and directories of the YousrtoreThemePackageFiles to your existing store directory. This will replace some files in your existing store.
  4. Login to you Store Admin Panel and you will see two alert messages on top of your store admin panel as shown in the image below. Install the Related Products Module and Yourstore Template Settings by clicking on Click to Install link.


  5. Once that is done you will see the Template has been installed successfully. message on top and will be redirected to the Theme Admin Panel.

Theme Installation for Zencart v1.5.5a and v1.5.5b

  1. Follow the Theme files installation steps as mentioned above
  2. Now find the Zencart v1.5.5a Theme Files or Zencart v1.5.5b Theme Files directory as per your version choice in the main downloaded package. Copy the content/files from this directory to your store existing store directory. This will replace some files in your existing store.

Theme Installation for Zencart v1.5.4

  1. Follow the Theme files installation steps as mentioned above
  2. Now find the Zencart v1.5.4 Theme Files directory in the main downloaded package. Copy the content/files from this directory to your store existing store directory. This will replace some files in your existing store.
