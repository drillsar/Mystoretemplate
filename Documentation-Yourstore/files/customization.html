Template Settings

YOURStore template can be customized according to customer's need and requirement from the Admin area. Template Settings makes customization very easy and hassle free.
Once the Template is installed completely, Template Settings menu can be found under Tools menu after login into template admin area.
Following are the features that can be customized using YOURStore Admin Template Settings.

General

General section under the YOURStore template settings covers following options :
  1. Theme Settings
  2. Font Settings
  3. Extra Settings
  4. Store Contact Details
  5. Newsletter Subscribe Details
  6. Enable/Disable Section
  7. Google Map
  8. Google Recaptcha Keys For Sign Up and Contact Us Page

Theme Settings

  • Theme Layout
  • Theme Color
  • Quick View
  • Display Additional Images Type
  • RTL Mode

  1. Theme Layout
    • The default theme layout is set to Default Layout, i.e the Light Layout. The dark layout if selected, your store will have the dark background on all your store pages. Live of which can be seen here
    • Click on Save Settings button and the changes will be applied to the front of store.
  2. Theme Color
    • The default theme color is set to #1FC0A0, you can enter the color code of your choice or select the color from the color picker provided.
    • Click on Save Settings button and the changes will be applied to the front of store.
  3. Quick View
    • This option lets you enable/disable the Quickview plugin integrated with the theme.
  4. Display Additional Images Type
    • This option comes with 3 different values. Selecting default option will remove the additional images slider from the products and will have a simple view.
    • Hover effect will have only single image effect on hover of any product. Live demo of this product hover style can be seen on our live demo version 1.
    • If you want to display multiple additional images (max 5 images) in slider view, it can be done by selecting Slider view option. Once the Slider view option is selected, another option below it will appear to let you choose the number of additional images.
  5. RTL Mode
    • Yes, the theme does support RTL and we have made it easy for users to switch their store to RTL mode. Select Yes or No based on your requirement and your whole store will be switched to the RTL.

Font Settings

  • General Font Family
  • Banner Caption Font Family
  • Character Set: Latin Extended
  • Custom Character Subset
  • Theme Font Size
  • Button Size

  1. General Font Family
    • YOURStore comes with 500+ Goolge fonts preloaded. Select the choice of your fonts and click on the Save settings button. The General Font family will affect all the fonts in theme excluding the banner cations that you have added.
  2. Banner Caption Font Family
    • With this option you can set the font family for banner caption separately. This will affect all the banners including the slideshow banners captions.
  3. Character Set: Latin Extended
    • Only selected fonts support extended character sets. For a complete list of available fonts and font subsets please refer to Google Web Fonts.
  4. Custom Character Subset
    • Only selected fonts support character sets. For a complete list of available fonts and font subsets please refer to Google Web Fonts. eg: greek,greek-ext
  5. Theme Font Size
    • The theme font size can be changed to either Small, Medium and Large.
  6. Button Size
    • Button size option changes the overall size of buttons used in the theme. The default size of button is large, which can be changed to Small and Medium size as per your requirement.

Extra Settings

  • Page Loader

  1. Page Loader
    • With this option you can disable the page loader which is by default integrated with the theme. Also if in case you want your own loader, just select the "Custom" option and that will let you upload your own "gif" image for loader.

Store Contact Details

  • Enter your Store contact details here. These details will be displayed in the Footer and the Contact us page.
  • If any detail is not required, leave the text-box empty and it will be removed from the store front.


Newsletter Subscribe Details

  • Managing Newsletter has been made easy with Mailchimp integration in YOURStore. New users can signup for free Mailchimp Services, while the already registered users can create their Signup forms and copy paste the code here.

  • After creating the List successfully, click on the List name, this will take you to the details of particular list.
  • Then click on Signup Forms Tab, this will open the details of different types of forms.
  • Click on "Embedded Forms options", this will open the "Embedded Form Code" detail with the preview of different forms.
  • Click on "Super Slim" option and copy the code below it. Paste the code in the text-area provided, as shown in image below.

  • The newsletter text box added by you will be displayed in both footer and the popup box.


Enable/Disable Content

  • YOURStore template comes with the Newsletter Popup box. It can be disable by selecting the value of Display Popup to No.
  • Also the Newsletter section can be disabled by selecting the value of Display Newsletter Section to No.
  • By default the value for both these sections has been set to Yes

Home Page

Home Page section under the YOURStore template settings covers following options :
  1. Homepage View
  2. Random Products Slider
  3. Brands Slider, Info Boxes and Testimonials Section

Homepage View

  • Homepage View has the option to set the layout of homepage to either fullwidth or 2 column layout. By default the homepage layout is set to Full Width.
  • If the Two Columns layout is selected further settings for Two Column layout will be displayed as seen in the image below :
  • The further options are for the Sidebar category megamenu that gets displayed on home page in the Two Column Layout homepage.
  • Following is the Sidebar Category menu with Theme Color on left and with Default color on right. The theme color can be set from the General Section as explained in the above steps.

Random Products Slider

  • The Random products slider option displays the products from your store randomly and displays in either the Slider view or the default Grid View as per your choice.
  • The status is disabled by default which can be rnabled as per your requirement.
  • The Random Products Layout is set to Slider View by default, which will look as the following image with 4 products in a row.
  • When the Random Products Layout is set to Default Grid View, which will look as the following image with 4 products in a row.
  • Using the Default Grid view you can control the number of random products to display using the admin option. For that go to Admin > Configuration > Maximum values and set the value of Random Products Module as per your choice. By default it is set to 8.
  • Also you can control the number of products to display in a row. But it should be in the multiple 12, as the theme uses Bootstrap. Preferably the value must be set to either 3 or 4 for better theme layout. This value can be changed from Admin > Configuration > Index Listing and set the value of Random Products Columns per Row.
  • The Random Products Image height and width can be changed from Admin > Configuration > Images. Find the Image - Random Products Width and Image - Random Products Height to change the default values.

Brands Slider, Info Boxes and Testimonials Section

  • Brands Slider
  • Info Boxes
  • Testimonials Section
  1. Brands Slider
    • Select Yes or No as per your requirement to display the brands slider on your store home page.
    • It comes with 3 different layout style, one with the white background, second with the theme background color that you set in the previous step and the last one is with grey background.
    • By default the Brands Slider is set to Display Yes and Display Style - 1.
    • Click on Save Settings button and the settings will be applied to the front of store.
  2. Custom Info Boxes
    • Info Boxes are the blocks used to add the custom text and icons that proves useful for an ecommerce store.
    • Select Yes or No as per your requirement to display the info boxes on your store home page.
    • It comes with 3 different layout style, one with the fixed 3 column layout or it can also be customized to 4 column layout and the second with unlimited columns slider layout, you can add as many number of columns as per your requirement and it will automatically be transformed in the slider.
    • The 3rd layout works only if Infoboxes and Testimonials section are set to Display Style 3 and the Latest News box is set to Display Style 2. For configuring the News Box see the News Box section.
    • By default the Info Boxes is set to Display No.
    • Click on Save Settings button and the settings will be applied to the front of store.
    1. Edit Custom Info Boxes
      • To edit the Custom Info Boxes text first login to your store Admin Panel and select the language of your choice (if your store is multilingual).
      • Now go to Admin > Tools > Define Pages Editor and select the define_custom_infoboxes_v1.php file from the dropdown menu if the display style is set to 1 for Custom Info Boxes.
      • The text area to edit will look something like as shown in image below :



      • The highlighted code shows you the Text which you need to edit, do not edit the HTML tags if you are not pro with the HTML code writing. Leave it as it is and just change the custom text as per your requirement.
      • Similarly for Custom Info Boxes Display Style 2 and Display Style 3, select and edit the define_custom_infoboxes_v2.php and define_custom_infoboxes_v7.php files respectively from the dropdown menu.
      • Click on Save button at the bottoma and the changes will be applied to the Store Front.
  3. Testimonials Section
    • The Testimonials are displayed in 3 different styles on home page, select the version of your choice.
    • There is also an option to add the custom background image for Testimonials Display Style - 1. You can upload the background image as per your requirement.
    • Click on Save Settings button and the changes will be applied to the front of store.
    • Also please note that you need to approve the testimonials added by your customers first in order to make it visible on the store front. To approve/edit/delete the testimonials go to Tools > Testimonials Manager, which will look something like below image. You can edit/delete/enable/disable the testimonials as per your requirement.



    • You can also add the testimonial by yourself, click on New Testimonial button and fill up the details asked.
    • General Testimonial settings can be customized from Admin > Configuration > Testimonials Manager section, as shown in the image below :

Simple Menu Settings

  • The main categories of store is contained under the SHOP menu.
  • Brands menu contain the list of manufacturers added by your from the Store Admin.
  • Ez-pages contains the Ez-pages links that are enabled in header section. Look here Admin > Tools > Ez-Pages for more info on Ez-Pages.

Sidebar Banners

With YOURStore you can add custom sidebar banners in Slider Style. The sidebar banner is a custom Sidebox which can be enabled or disabled from Admin > Tools > Layout Boxes Controller. Look for box name pzen_sidebar_banner.php and you can change the postion of sidebox too as in normal Zencart sidebox process.


  • Using the Sidebar Banner section under them admin panel, you can add banners as per your requirement. Internal/External link can also be added for the particular banner.
  • Sort order of banners can be changed using simple drag and drop.

Products Display Style

Using these options the New Products, Featured Products, Special Products and Best Seller Products diplay style can be modified to either Slider view or the default grid view.
