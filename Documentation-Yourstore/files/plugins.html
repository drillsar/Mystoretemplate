Zencart Plugins

We have used the following Zencart plugins in YOURStore. All the credit goes to their respective owner/developer.

CSS Horizontal Dropdown Menu

It is used to get the pages dynamically from the backend as you create it. You can find it from here..

Zen Lightbox

It is used to view the product image on click. The image will be zoomed on click. You can find it from here..

Language Currency Links in Header

It is used to get the language and currency. You can find it from here..

Ajax Add to Cart

It is used to add a product to cart using AJAX (that means no page reload) and will display a nice popup - shopping cart slider - showing cart contents. You can find it from here..

Compare Multiple Products

This contribution adds a comparison system to the product listing page. It is done through a session so that products can be compared across multiple categories. You can find it from here..

Wishlists

This allows registered users to "save" products in personal wishlists. You can find it from here..

Column Layout

Product Listing reworked to show in a grid format (columnar) instead of just rows. You can find it from here..

Image Handler

Image Handler 5 is really meant to ease the management of product images (particularly the management of additional product images), and to help improve page performance by optimizing the product images. You can find it from here..

dgReviews

This Plugin is used to view product reviews on product info page. You can find it from here..

Disqus Comment Form

This module give your site a capability to create a comments form on product info page. More information can be found from here..
